# personal-space

## 安装依赖
```
yarn install
```

### 客户端启动 http://localhost:8080/#/
```
yarn client
```

### 启动数据库（mongoDB）mongodb://localhost/PERSONAL_SPACE
```
sudo mongod
重启mongo
```

### 新建一个终端窗口，服务端启动 http://localhost:3000/#/
```
yarn server
```

### 技术栈
```
1、client: vue-cli + vue2.0 + vuex + vue-router + axios + element-ui
2、server: nodejs
3、db: mongodb
```
